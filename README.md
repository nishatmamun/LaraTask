
# Laravel Task

A simple project of fetching user details from API.


## Deployment

To deploy this project

-> Create a Database and name it LaraTask.

-> Php artisan migrate

-> npm install && npm run dev 

-> php -S localhost:8000 -t public/

-> Import data before login.




## API Reference
